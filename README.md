## Description

**Evid.exe** est une application mobile conçue pour améliorer la compréhension de concepts scientifiques souvent perçus comme abstraits.

L’application transforme des notions théoriques en **visualisations interactives** que l’utilisateur peut manipuler en temps réel afin de mieux comprendre les relations entre les variables et les phénomènes étudiés.

---

## Fonctionnalités principales

L'application est divisée en trois modules principaux :

### Mathématiques
Visualisation de concepts liés au calcul différentiel et intégral.

Exemples :
- intégrale définie
- sommes de Riemann
- aire sous une courbe
- approximation numérique

### Physique
Simulations interactives de phénomènes physiques.

Exemples :
- mouvement des projectiles
- lois de Newton
- travail et énergie
- conservation de l’énergie

### Programmation
Visualisation du fonctionnement de structures fondamentales de programmation.

Exemples :
- variables
- structures conditionnelles
- boucles
- logique algorithmique
